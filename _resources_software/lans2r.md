The `Look@NanoSIMS` (short LANS) Matlab module written by Lubos Polerecky (*Polerecky L., Adam B., Milucka J., Musat N., Vagner T. and Kuypers M. M. M. (2012). Environ. Microbiol. 14, 1009â€“1023.*) makes it easy to process NanoSIMS data and draw regions of interest (ROIs). The **lans2r** package provides a convenient interface to import ion maps and ROI data generated by LANS for processing and plotting in R/RMarkdown. Additionally, it can process NanoSIMS tuning (HMR) files.
